Meu Portfólio Profissional em React
Portfólio profissional em React — páginas, componentes reutilizáveis, deploy. 

📁 Mapa do Site (Páginas)
* Home: Página inicial. * Projetos: Exibição de trabalhos realizados. * Contato: Informações para contato profissional. 

🧩 Componentes Principais
* Header: Cabeçalho de navegação. * Footer: Rodapé com créditos. * ProjectCard: Cartão para exibição de projetos. 

🚀 Como Executar
1. Instale as dependências: npm install 2. Inicie o servidor de desenvolvimento: npm run dev 

🤝 Como Contribuir
Faça um fork do projeto, crie uma branch para sua feature e envie um Pull Request.
